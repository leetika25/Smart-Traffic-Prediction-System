🚦 Smart Traffic Prediction System using Machine Learning
📌 Description

A Machine Learning-based system that predicts traffic congestion levels using vehicle count data such as cars, bikes, buses, and trucks. The model classifies traffic into Low, Normal, High, and Heavy categories using a Random Forest algorithm.

🎯 Objectives
Predict traffic conditions based on vehicle count
Apply machine learning for real-world problem solving
Analyze and visualize traffic patterns

🧠 Technologies Used
Python 🐍
Pandas
Matplotlib
Scikit-learn
Random Forest Classifier

📊 Dataset
Format: CSV file
Features:
CarCount 🚗
BikeCount 🏍️
BusCount 🚌
TruckCount 🚚
Target:
Traffic Situation (Low / Normal / High / Heavy)

⚙️ Project Workflow
Load dataset
Data preprocessing (cleaning & transformation)
Feature selection
Train-test split
Model training (Random Forest)
Model evaluation (accuracy)
Prediction & output

📈 Output
Displays model accuracy
Predicts traffic level based on input
Shows traffic distribution graph

🚀 Features
Easy to use
Accurate prediction model
Data visualization included
Model saved for future use
